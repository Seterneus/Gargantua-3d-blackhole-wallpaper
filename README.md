# Gargantua 3D — Relativistic Black Hole Live Wallpaper & WebGL Engine
**Created by Abdullaev Samandar (`Seterneus`)**

A pure, highly optimized 3D raymarched Gargantua Black Hole featuring accurate Schwarzschild gravitational lensing, relativistic Doppler beaming, multi-layered photorealistic stars, swirling accretion plasma, and a procedural dark cosmic ambient soundscape.

## ✨ Key Features

- **Relativistic Gravitational Lensing**: Physically inspired light deflection around an event horizon with Einstein ring distortion.
- **Photorealistic Milky Way, Spiral Galaxy & Planetary Eye Nebula**: Deep cosmic background featuring a sleek 5% Milky Way galactic band, dark Great Rift dust lanes, localized Hydrogen-Alpha crimson emission wisps, a distant Andromeda-style spiral galaxy, and a compact Planetary Emerald Ring Nebula lensed into Einstein arcs behind Gargantua.
- **Relativistic Doppler Beaming Asymmetry**: Adjustable slider controlling the cinematic brightness & color shift of approaching vs. receding accretion plasma.
- **Infalling Photon Sparks & Embers**: Glowing cosmic sparks orbiting near the photon sphere that spiral inward across the event horizon.
- **7 Dynamic Color Palettes**:
  - `Interstellar Gold (Reference Image)`
  - `Quasar Electric Cyan`
  - `Void Magenta & Purple`
  - `Crimson Inferno Red`
  - `Pure Emerald Green`
  - `Aurora Borealis Green (Northern Lights)`
  - `Aurora Violet (Geomagnetic Solar Storm)`
- **Procedural Dark Deep Space Soundscape**: Built-in real-time Web Audio synthesis featuring deep sub-bass gravitational drones and haunting interstellar minor pad chords.
- **Audio Reactivity**: Accretion disk energy and photon ring pulse dynamically to music or ambient sounds.
- **Analytical Edge Anti-Aliasing**: Sub-pixel FXAA-style edge feathering around the event horizon for razor-sharp rendering.
- **Interactive 3D Camera**: 360° orbital drag controls and mouse-wheel zoom.

## 🎮 Controls & Hotkeys

| Key / Action | Description |
| :--- | :--- |
| **Click & Drag** | Orbit 360° around the black hole |
| **Mouse Wheel** | Zoom in / out |
| **`C` Key** | Cycle through all 7 color palettes |
| **`M` Key** | Toggle procedural dark cosmic soundscape Mute/Unmute |
| **`Spacebar`** | Toggle Automatic Orbital Circling |
